### 🍎 GOLDEN ENGINE
Golden Engine was originally created as a specialized recreation of the Golden Apple Edition, intended to be a fix for the "Codename Bullshit" and vanilla version issues. It aims to be the ultimate, inescapable, and most chaotic alternative for newbie coders.
### ⚠️ BEWARE: EYESORES ARE PRESENT
This engine is designed to be visually intense. If the screen isn't flashing and the window isn't shaking, you aren't doing it right.
### Installation:
Refer to the Build Instructions (Uses Haxe 4.3.2 and HMM).
### Customization:
If you wish to disable things like Rainbow Eyesores, 3D Objects, or Discord Pings, you can refer to the Project.xml file.
Inside Project.xml, you will find several variables to customize Golden Engine to your liking. Disabling features is simple: just delete the line or comment it out by wrapping the line in XML comments, like this: <!-- YOUR_LINE_HERE -->
LUA_ALLOWED: Comment this out to break mod scripts.
3D_OBJECTS: Disables the 3D models button and Dave/Bambi renders.
CLOSEABLE_GAME: COMMENTING THIS OUT WILL PREVENT THE USER FROM EXITING THE GAME TO SCARE THEM :3
### Softcoding (.lua/.hx)
Golden Engine supports both Lua and HScript (Iris). You can learn how to use the custom Golden functions (like window shaking) in the wiki.
